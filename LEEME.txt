═══════════════════════════════════════════════════════════════
  🎯 INICIO RÁPIDO
═══════════════════════════════════════════════════════════════

1️⃣  Instalar dependencias
    $ npm install

2️⃣  Configurar variables de entorno
    $ cp .env.example .env
    (Editar .env con datos reales)

3️⃣  Configurar base de datos MySQL
    (Ver README.md para script SQL)

4️⃣  Iniciar servidor
    $ npm start

5️⃣  Abrir en navegador
    http://localhost:4000

═══════════════════════════════════════════════════════════════
  🌐 DESPLIEGUE EN VPS
═══════════════════════════════════════════════════════════════

1️⃣  Subir proyecto al VPS
2️⃣  Instalar dependencias (npm install)
3️⃣  Configurar .env con datos reales
4️⃣  Configurar MySQL y crear tablas
5️⃣  Configurar Nginx (ver nginx.conf.example)
6️⃣  Iniciar con PM2 (pm2 start ecosystem.config.js)
7️⃣  Configurar SSL con Certbot

Ver README.md para instrucciones detalladas
